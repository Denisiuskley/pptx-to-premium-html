# PPTX to Premium HTML Converter

Модульный конвейер для автоматической конвертации презентаций PowerPoint (.pptx) в современные анимированные HTML-файлы с поддержкой формул (MathJax), динамического расположения элементов и ИИ-анализа текста.

## Особенности

- **Высокая точность**: Сохранение геометрии слайдов, таблиц и формул.
- **Премиальный дизайн**: Современный темный интерфейс с использованием CSS-градиентов, стеклянного морфизма и плавной анимации (GSAP).
- **Интеллектуальный анализ**: Использование LLM для автоматического структурирования выводов и планов исследований из текстовых документов.
- **Автономность**: Весь результат вшивается в один HTML-файл (картинки, шрифты, скрипты).
- **Адаптивность**: Динамический рендеринг формул и текста в зависимости от плотности контента.

## Структура проекта

- `src/` — Исходный код модульного пайплайна.
    - `core/` — Парсинг PPTX и базовый цикл обработки.
    - `converters/` — Конвертеры контента (HTML, MathML).
    - `utils/` — Вспомогательные функции (изображения, ИИ, кэш).
- `data/` — Входные файлы (презентации, текстовые выводы).
- `logo/` — Логотипы для оформления.
- `libs/` — Локальные JS-библиотеки.
- `main.py` — Точка входа в программу.

## Установка

1. Склонируйте репозиторий.
2. Создайте виртуальное окружение:
   ```bash
   python -m venv venv
   source venv/Scripts/activate  # для Windows
   ```
3. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
4. Скачайте необходимые JS-библиотеки:
   ```bash
   python download_libs.py
   ```

## Использование

1. Поместите вашу презентацию в папку `data/`.
2. Настройте файл `.env`, добавив `OPENROUTER_API_KEY` (если нужен ИИ-анализ).
3. Запустите конвертацию:
   ```bash
   python main.py
   ```
   По умолчанию будет обработан файл `data/Промежуточная.pptx`. Если вы хотите обработать другой файл, укажите его путь:
   ```bash
   python main.py data/my_presentation.pptx
   ```
